# QUICK START - Compile Instructions

## Unzip and Compile

```bash
# 1. Unzip the package
unzip Emergent_compile_ready.zip

# 2. Navigate to the directory
cd Emergent

# 3. Compile (choose one option)

# Option A: Quick compile (no bibliography)
xelatex main.tex

# Option B: Full compile (with bibliography)
xelatex main.tex
biber main
xelatex main.tex
xelatex main.tex
```
